## About Me

I am **Abhishek**, a Computer Science student pursuing an **Honors specialization in Data Science**, and a Java Backend Development intern at **SyntecxHub**. I'm deeply interested in clean software architecture, exception-driven design, and building systems that are robust from the ground up. I enjoy taking a problem, breaking it into well-defined layers, and writing code that's easy for the next person to read and extend.

My work sits at the intersection of **backend engineering**, **data science**, and **open source contribution**. I like designing layered Java applications, contributing fixes to real-world tooling used by thousands of developers, and exploring how data-driven techniques can complement solid backend systems.

## Personal Note

I'm currently focused on building a strong portfolio of backend-first Java projects — each one built without external libraries, with custom exception hierarchies and deliberate data structure choices. I believe good code is only as good as the discipline behind it, and I try to make every project I ship readable, maintainable, and something I'd be comfortable handing to a team.
